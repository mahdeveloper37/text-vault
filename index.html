WHAT IS TEXT ADVENTURE?

Text adventure games (also called interactive fiction) are one of the oldest forms of computer games. You explore a world, solve puzzles, and tell your story entirely through typed commands and text descriptions.

Pioneered in the 1970s by games like ADVENTURE (Colossal Cave) and popularized by Infocom titles such as ZORK, Hitchhiker's Guide to the Galaxy, and Planetfall — this is where gaming began.

COMMANDS

Type commands in plain English. The parser understands verb + noun combinations:

GO NORTH / N / S / E / W / U / D
LOOK — describe current room
EXAMINE [item] — inspect something
TAKE [item] — pick up an object
DROP [item] — drop from inventory
INVENTORY / I — list carried items
OPEN / CLOSE [thing]
USE [item] — use an item
READ [item] — read text
HELP — show this list
RESTART — start over
QUIT — exit game

TIPS FOR BEGINNERS

• Draw a map as you go. A pencil and paper is your best friend.

• EXAMINE everything you find — even seemingly mundane objects.

• Save often (type SAVE). Some games have unwinnable states.

• READ anything with text — signs, notes, books, inscriptions.

• If stuck, try every direction, and every verb you know.
